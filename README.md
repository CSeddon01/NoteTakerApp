## 🌟 The Note Taker Application

# 🧐 Details 

Using starter code, create a notes page that allows you to see current notes, enter new notes, and delete notes.  This application will be deloyed with Heruko.

# Built with:

* HTML 
* JavaScript
* CSS
* Bootstrap
* Express.js
* Insomnia
* Heruko

# Deployment 

* node.js, express.js
* Uniqid package
* npm install
